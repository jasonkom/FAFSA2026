The 7 Biggest FAFSA Changes for 2026–27 (Congress Made Permanent in 2024)

  1. Expected Family Contribution (EFC) is dead → Replaced by Student Aid Index (SAI)
  2. Negative SAI allowed down to –1,500 → More aid for the lowest-income families
  3. 100% family farm & small-business exclusion → Net worth no longer counted
  4. Pell Grant maximum increased to $7,895 (from $7,395)
  5. Only ~36 questions instead of 108
  6. Every contributor must create an FSA ID
  7. Mandatory IRS Direct Data Exchange → No more manual 1040 uploads

Frequently Asked Questions (Answered for 2026–27)

When does the 2026–27 FAFSA open?

October 1, 2025 at 8:00 AM EST. The form is already in final testing.

Do grandparents’ 529 plans still hurt aid?

No — starting 2026–27, grandparent 529s are no longer reported on the FAFSA (only on CSS Profile for private colleges).

Do I still need to list siblings in college?

No — the “number in college” question is permanently removed. Aid is now calculated per student.
